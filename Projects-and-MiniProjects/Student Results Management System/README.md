# Student Marks Management System (Core Java)

A simple console-based Java program that evaluates student performance using basic Java concepts such as arrays, conditionals, and methods.

## Features
- Stores marks of multiple students using a 2D array
- Calculates total marks and average per student
- Determines pass or fail status based on subject-wise cutoff
- Assigns grades based on total marks
- Displays a formatted result summary for each student

## Concepts Used
- 1D and 2D Arrays
- Conditional Statements (`if-else`)
- Loops
- Methods
- Basic Input / Output

## Grading Logic

|------------|-------|
| > 500      | A     |
